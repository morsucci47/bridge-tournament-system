5) Modificare dbConnessione.php per le credenziali di accesso al database
	// VARIABILI DI SISTEMA
	$home_pages="http://mio-sito.it";
	$home_proc="http://mio-sito.it/BridgeCoppieR2";
	$home_archive= "Archivio";

	// CREDENZIALI DATABASE
	// nome di host
	$host = "localhost";
	// username dell'utente in connessione
	$user = "MY-USR";
	// password dell'utente
	$password = "MY-PSWD";
	// nome del database
	$db = "my_database";


6) Eseguire la procedura:
